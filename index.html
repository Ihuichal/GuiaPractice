android

Desarrollo

● Se utilizó Android Studio como entorno de desarrollo.
● Lenguaje de programación JAVA

group

Gestión

● Se empleo Scrum para trabajar colaborativamente.
● Para realizar la gestión de trabajo y avance, contamos con Jira Atlassian.
● Bitbucket se uso como repositorio, para almacenar los avances

cloud

Servicios

● Empleamos servicios de Amazon Web Services, para conectar la aplicación a un servidor y con ello poder almacenar y manejar datos.
● Lambda, API Gateway y DynamoDB son los servicios de AWS que se ocupan actualmente
